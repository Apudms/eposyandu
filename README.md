# 🧑‍💻 Setup

Clone the repo and install dependencies:
```bash
git clone https://github.com/Apudms/eposyandu.git
cd eposyandu
composer update
# Edit your .env file and all the credentials
# Import the existing database in the Project folder
php artisan serve


```
